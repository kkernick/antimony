# Binaries

This folder contains all the binary targets of Antimony; there are three categories:

1. `antimony` itself
2. Install utilities that Antimony uses when installed on a system; these are delineated with a `-`, such as `antimony-dumper`.
3. Crate utilities that are used to aid in the development of the project, and are not included in a system installation. These are delineated with a `_`, such as `antimony_bench`
